# 🚀 多機能 LINE メディア保存 Bot (GAS/clasp)

LINEで送信された画像、動画、ファイルをGoogle Driveに自動保存し、スプレッドシートで一元管理する高度なボットです。
2026/03/19の大規模アップデートにより、セキュリティとパフォーマンスが大幅に向上しました。

## ✨ 主な機能

### 📂 メディア自動保存 & グループ別ログ
- **自動フォルダ分け**: グループ/ユーザーごと、かつ日付（YYYYMMDD）ごとにフォルダを自動生成。
- **独立ログシート（New!）**: **グループごとに専用のスプレッドシートが自動作成**されます。他のグループのログが混ざることはありません。
- **共有設定**: 作成されたログシートは「リンクを知っている全員（閲覧のみ）」に自動設定され、チャットメンバーで共有可能です。

### 📝 メモ機能 (/memo)
- /memo [内容]: 送信したメディアに対して、ログシートの「内容/memo」列にテキストを記録できます。

### 🏷 高度なリネーム (/rename)
- **通常**: /rename [名前] - 直前に保存したファイルをリネーム。
- **一括（New!）**: /rename last [個数] [名前] - 直近N件のファイルに連番を付けて一括リネーム。
- **全件**: /rename all [名前] - 本日の全ファイルを一括リネーム。

### 🗑 ファイル削除 (/delete)
- /delete: 直前に保存したファイルを即座にゴミ箱へ移動。
- /delete [番号]: /list で表示された番号を指定して削除。

### 📊 閲覧・管理
- **/list**: 本日保存したファイル名の一覧を新着順に表示。
- **/link**: 保存先フォルダと専用ログシートのURLを表示。
- **/stats**: 保存件数や待機中のログ件数などの統計を表示。

## ⚙️ システム仕様

### ⚡️ 高速レスポンス & 非同期ログ
- **Reply-First**: LINEへの返信を最優先し、重いドライブ保存やログ書き込みはバックグラウンドで処理します。
- **15分バッチ（New!）**: ログの書き込み負荷を抑えるため、15分ごとにまとめてスプレッドシートへ書き込みます。

### 🔒 セキュリティ
- **管理者用シークレットログ**: すべてのチャット活動（テキスト含む）を管理者が指定した専用シートに別途記録。
- **権限管理**: /contact コマンドで開発者（管理者）へ直接メッセージを送信可能。

## 🛠 技術スタック
- **Runtime**: Google Apps Script (V8 engine)
- **Tool**: clasp (Command Line Apps Script Projects)
- **API**: LINE Messaging API
- **Storage**: Google Drive, Google Sheets
